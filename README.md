# npb-defense-analysis

## 🎯 プロジェクト概要

「守備で交代したばかりの選手にはボールが飛びやすい」という野球界の定説（あるある）を、2023年プロ野球の1年間の全試合データを用いて統計的に検証したプロジェクトです。

直感的な経験則を定量的なデータで裏付け、特定の条件下における事象の発生確率が統計的に有意に異なるかを分析しました。

## 🚀 技術的ハイライト

* **データ収集（Web Scraping）**: オープンデータが存在しなかったため、Pythonを用いてスポーツニュースサイトから2023年度の全試合経過データをスクレイピングし、独自のデータセットを構築しました。
* **統計的仮説検定**: カイ二乗検定（Chi-square test）を用い、通常時の打球飛来率と守備交代時の飛来率の差が統計的に有意であることを証明しました。
* **データ集計パイプライン**: 投手・捕手を除く7ポジションの打球飛来機会を定義し、通常時（7〜20%）と交代時（24%）の確率差を定量化しました。
* **AI共創分析**: ChatGPTを分析パートナーとして活用し、仮説の立案、スクレイピングコードの実装、統計手法の選定、および結果の解釈までを一貫して高速化しました。

## 📊 分析結果サマリー

* **守備交代があった選手の打球飛来率**: **24%**
* **通常時の各ポジションの打球飛来率**: **7% 〜 20%**
* **結論**: 統計的検定の結果、守備交代直後の選手には通常時よりも有意に高い確率で打球が飛ぶことが確認されました。
